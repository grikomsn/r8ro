# Support

Use [GitHub Discussions](https://github.com/grikomsn/r8ro/discussions) for
setup, usage, and self-hosting questions.

Use the [issue tracker](https://github.com/grikomsn/r8ro/issues) for
reproducible bugs and feature requests. Search existing issues before opening
a new one and include the requested environment and reproduction details.

Security vulnerabilities must be reported privately as described in
[SECURITY.md](SECURITY.md).
